# Warm Burnout for iTerm2

Your terminal was also frying your corneas. The burnout is spreading.

## Install

Double-click either `.itermcolors` file to import it directly, or import manually:

1. Open iTerm2
2. Go to **Settings** > **Profiles** > **Colors**
3. Click **Color Presets...** > **Import...**
4. Select `Warm Burnout Dark.itermcolors` and/or `Warm Burnout Light.itermcolors`

## Configure

After importing, select the preset from the **Color Presets...** dropdown:

- **Warm Burnout Dark**: for the 3am sessions
- **Warm Burnout Light**: for when someone forces you to open the blinds

## Verify

Run `ls --color` or any ANSI color test script. Warm browns instead of searing blues.

## Palette

Both themes derive from the canonical Warm Burnout palette defined in the root [`AGENTS.md`](../AGENTS.md). The ANSI colors match the Ghostty terminal theme exactly.
